# Loan Prediction System 🚀

A Machine Learning-powered web application that predicts both **Loan Approval Status** and **Loan Amount** using multiple Machine Learning algorithms. The project allows users to compare model performance and make real-time predictions through an interactive web interface.

## Features

* ✅ Loan Approval Prediction
* 💰 Loan Amount Prediction
* 📊 Multiple Machine Learning Models
* 🔄 Model Comparison
* 🌐 Flask-Based Web Application
* 🚀 Deployed on Vercel
* 📈 Real-Time Predictions

## Machine Learning Models

### Classification Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

### Regression Models

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## Model Performance

### Classification Accuracy

| Model                    | Accuracy |
| ------------------------ | -------- |
| Logistic Regression      | 79.86%   |
| Decision Tree Classifier | 97.78%   |
| Random Forest Classifier | 97.78%   |

### Regression Performance (R² Score)

| Model                   | R² Score |
| ----------------------- | -------- |
| Linear Regression       | 0.8514   |
| Decision Tree Regressor | 0.7128   |
| Random Forest Regressor | 0.8457   |
